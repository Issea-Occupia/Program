# 🔥 **第 1 关（基础构造）——写一个最小可用 Array 类**

**要求：**

1. 使用模板，让它能存任意类型：`Array<T>`
2. 构造函数接受一个 size：`Array<int> a(5);`
3. 能动态分配一块 T 类型的数组
4. 提供 operator[] 用于读写元素
5. 提供 size() 返回长度
6. 析构函数正确释放内存

**示例用法（你的代码必须能支持）：**

```
Array<int> a(5);
a[0] = 10;
a[1] = 20;
std::cout << a[1];  // 输出 20
std::cout << a.size(); // 输出 5
```

**提示：你必须包含这几个成员：**

- T* data;
- size_t n;

**不需要考虑拷贝构造、不需要考虑移动、不需要考虑异常安全、不需要 operator+。
 这一关完成最基础的结构即可。**
